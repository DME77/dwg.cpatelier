🔗 Connect Google Drive
Enter your Google OAuth 2.0 Client ID to enable automatic syncing to Google Drive. Once connected, every drawing gets a folder under the linked GFC Drive folder, organized as Discipline Tab → Drawing → Coordination / GFC Submissions — coordination status and GFC status updates are logged there automatically as they happen.
1. Go to Google Cloud Console → Credentials
2. Create an OAuth 2.0 Client ID (Web application type)
3. Add this portal's URL to Authorized JavaScript origins
4. Enable the Google Drive API in the project
5. Paste the Client ID below and click Connect

＋ Add Discipline Tab

This creates a brand-new, empty discipline register — its own tab, drawing cards, status tracking, coordination/GFC log, dashboard stats, and Google Drive folder tree — exactly like Arch/MEP/Structure.

🔐 Drawing Access Control

Choose who automatically gets Editor rights on Drive, at any level: a default list for everything, a list per project tab (covers that whole project), and a list per discipline tab (Arch/MEP/Structure and any custom tabs). Grants are applied when the matching folder is created; everything inside inherits them. Everyone else keeps view-only access.

Tip: leave a discipline blank to use only the default list. Contacts from your directory appear as suggestions.

✏️ Edit Drawing

Tracker history moves with the drawing. If a Drive folder was already created under the old ID, a new folder will be used for the new ID on the next upload — existing files stay in the old folder.

＋ Add Zone

Zones divide this project's drawing register. Drawings added while a zone is selected are tagged to it automatically; you can also change a drawing's zone from its ✎ edit form.
